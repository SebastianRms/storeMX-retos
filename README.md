StoreMX - E-commerce Escalable & Seguro
Descripción del Proyecto
Desarrollo de una solución integral de E-commerce diseñada para gestionar operaciones de almacén, ventas y movimientos de inventario de forma intuitiva y segura. Este proyecto demuestra mi capacidad para construir aplicaciones Fullstack robustas, conectando una interfaz de usuario dinámica con un servidor seguro.

Stack Tecnológico
Frontend (Angular)
Estado y Reactividad: Implementación de RxJS con patrones de Observables para un manejo de estado eficiente y asíncrono.

Seguridad de Rutas: Uso de Guards para proteger el acceso a módulos sensibles.

Lógica de Negocio: Centralización de funciones avanzadas mediante Servicios inyectables.

Gestión de Sesiones: Interceptors para la inyección automática y manejo de tokens de autenticación.

Control de Roles: Directivas personalizadas para la validación de perfiles (Admin/User) y visualización dinámica de ofertas.

Backend (Node.js & Express)
Seguridad: Encriptación de datos sensibles con Bcrypt y manejo de sesiones mediante JWT (JSON Web Tokens).

Arquitectura: Estructura basada en Middlewares para crear barreras de seguridad y validación de peticiones.

Robustez: Sistema de manejo de errores diferenciado para entornos de Desarrollo y Producción.

Funcionalidades Clave
Operación de Almacén: Lógica diseñada para el control de movimientos de stock y ventas en tiempo real.

Escalabilidad: Código modular preparado para el crecimiento de la operación y volumen de datos.

Interfaz Intuitiva: Enfoque en la experiencia del usuario final para reducir errores de captura.

Sobre el Desarrollador
Soy un Desarrollador Fullstack en formación (T.S.U. en Programación) con 6 años de experiencia previa liderando almacenes y flujos logísticos en empresas como Club Necaxa y Zapata Camiones. Mi enfoque es resolver problemas reales de negocio a través de código limpio y arquitecturas seguras.
